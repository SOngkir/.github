## Tentang SOngkir

**SOngkir** adalah plugin Shopify yang dirancang khusus untuk merchant Indonesia. Kami menyediakan solusi pengecekan ongkos kirim domestik yang cepat, akurat, dan mudah diintegrasikan ke toko online Anda.

### Mengapa SOngkir?

- **Cepat & Real-time** - Tarif ongkir selalu update dari berbagai ekspedisi populer
- **Multi Kurir** - Mendukung JNE, J&T, SiCepat, AnterAja, Pos Indonesia, dan lainnya
- **Mudah Dipasang** - Instalasi dalam hitungan menit tanpa perlu coding
- **Support Lokal** - Tim support Indonesia yang siap membantu dalam Bahasa Indonesia

## Fitur Utama

| Fitur | Deskripsi |
|-------|-----------|
| Real-time Rates | Tarif ongkir langsung dari API kurir |
| Multi Kurir | 10+ kurir populer Indonesia |
| Checkout Integration | Terintegrasi dengan Shopify checkout |
| Dashboard Analytics | Pantau statistik pengiriman |
| Lokasi Fleksibel | Atur multiple lokasi gudang/toko |
